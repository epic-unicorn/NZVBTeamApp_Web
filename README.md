# NZVB Team App

A Flutter application for tracking Dutch futsal team information from the Nederlandse ZaalVoetbalBond (NZVB).

## ⚠️ Notice

**The app is no longer functional.** The NZVB API has been changed, which has broken the app's ability to fetch team data. The application requires an API update to work again.

## Features

- **Team Selection**: Choose your favorite football team and league
- **Rankings**: View current league standings and team positions
- **Results**: Check match results and scores
- **Schedule**: See upcoming matches and fixtures
- **Multi-platform**: Available on Android, iOS, Windows, and Web
- **Persistent Storage**: Save your selected team and league preferences

## Platforms

- **Android**: Available on Google Play Store
- **iOS**: Available on App Store
- **Windows**: Desktop application
- **Web**: Browser-based version
